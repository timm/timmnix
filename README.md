timmnix
=======

unix config utils

## Installation

```
cd 
git clone  https://github.com/timm/timmnix.git
cd timmnix
make install

```

Optinally, add to `$HOME/.bash_profile`:

```
. tnix
```

## After installation

Either

```
cd 
. tnix
```

or, if you editted `.bash_profile`, just log out and log in again.
